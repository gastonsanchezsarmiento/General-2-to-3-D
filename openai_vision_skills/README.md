# OpenAI Vision Skills Pack

This folder contains three lightweight skills for local Codex projects:

- `openai-model-calls.md`
- `vision-document-pipeline.md`
- `engineering-drawing-vision.md`

Reference files are in `skills/references/`.

The engineering drawing skill is intentionally separate from ordinary document vision because large-format technical drawings require higher DPI rendering, full-sheet context images, targeted high-resolution crops, 20% overlap tiling, and coordinate/evidence traceability.
